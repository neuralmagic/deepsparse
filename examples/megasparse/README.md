## <div>`INTRO`</div>

<samp>

<div>
MegaSparse allows you to quickly serve a streamlit app running with the DeepSparse Server as a backend to run inference on multiple transformer models. The present demo loads 18 Sparse BERT ONNX models fine-tuned for the question answering task. The purpose of this app is for you to familiarize yourself with the high performance inference speeds of sparse models.
</div>

## <div>`INSTALL`</div>

```bash
git clone https://github.com/<placeholder>/megasparse.git
cd megasparse
pip install -r requirements.txt
```
## <div>`START SERVER`</div>

Run the DeepSparse Server:
```bash
deepsparse.server --config_file server/config.yaml
```

This will download models from the `config.yaml` file and then start the DeepSparse Server on localhost `0.0.0.0` and port `5543`.

## <div>`START CLIENT`</div>

Open a new terminal and run Streamlit:

```bash
streamlit run client/app.py \
    --theme.backgroundColor blue \
    --theme.secondaryBackgroundColor black \
    --theme.textColor white \
    --browser.serverAddress="0.0.0.0"
```
This will start the Streamlit app in localhost `0.0.0.0` and port `8501`.
Visit `http://0.0.0.0:8501` in your browser to view demo.

### Support

- All models able fit on 16GB RAM of a c2-standard-4 VM instance on GCP
- Tested on Ubuntu 20.04.4 LTS
- Python 3.8.10
